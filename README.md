# Olist Customer Analytics

Customer Analytics project based on the Olist Brazilian E-Commerce dataset.

## Project objective

The goal of this project is to practice the complete analytics workflow using Power BI, including:

- ETL and data preparation.
- Customer volume analysis.
- RFM segmentation.
- Churn and retention analysis.
- Cohort analysis.
- CLV and customer value metrics.
- Simulated variables when needed.
- Interactive Power BI dashboard.

## Repository structure

- data/clean/: cleaned intermediate datasets.
- data/processed/: final tables prepared for Power BI.
- data/simulated/: simulated variables or auxiliary tables.

## Dataset

Dataset: Brazilian E-Commerce Public Dataset by Olist, available on Kaggle.

The original data files are not included in this repository and should be downloaded manually into data/raw/.
